# Unibotics Challenges

This repository will showcase the solutions and results of the various challenges presented by Unibotics. Specifically, it focuses on the challenge of autonomous driving by following a line and human rescue using a drone. The code is implemented in Python and runs directly in the browser by connecting to a Docker image.
